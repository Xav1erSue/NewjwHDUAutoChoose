# 杭电新教务系统自动学评教 油猴脚本

> 更新时间：2022 年 6 月 6 日
>
> 最新版本：V3.2.2
>
> 更新日志：修复比例设置问题

## 1. 实现功能：

- 跳过确认阅读界面
- 设置选项比例和提交方式
- 自动学评教

## 2. 使用步骤

- 加载脚本，进入学评教界面
- （可选）进行参数配置
- （必选）将显示课程数调大，使课程可以全部显示在列表中
- 点击开始按钮开始自动学评教
- 手动确认后提交

## 3. 参数说明

```js
const config = {
  /**
   * 是否开启直接提交
   * 默认为 false
   * 开启后点击开始按钮会直接打分并提交
   */
  startAndSubmit: false,
  /**
   * 选项比例，自动学评教时 A、B、C、D 的分别占比
   * 默认值为 { A: 0.6, B: 0.4, C: 0, D: 0 } ,即 A: 60%, B: 40%, C: 0%, D: 0%
   */
  ratio: {
    A: 0.6,
    B: 0.4,
    C: 0,
    D: 0,
  },
  /**
   * @todo 未实现
   * 白名单，单独设置老师的学评教参数
   * （似乎没有什么意义？如果有人希望有这个功能的话我再做吧）
   */
  whiteList: [
    {
      name: "张三",
      radio: {
        A: 0.5,
        B: 0.3,
        C: 0.1,
        D: 0.1,
      },
    },
  ],
  /**
   * 监听时间间隔，用于监听 DOM 节点是否已经挂载
   * 性能极差的设备上如果调太小可能会失败
   * 默认值为 200 ms
   */
  interval: 200,
};
```
